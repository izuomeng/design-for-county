# 宜君农民画包装 · 采集字段清单 + 提问流程

> 目标：Agent 在调用 `generateImage` **之前**，必须按本流程把字段收齐。
> 原则沿用 [SKILL.md](./SKILL.md)：让用户「指」不让用户「想」、凶狠默认值、3~4 步出图、Agent 先拟草稿。

---

## 一、字段清单（生图前必须齐）

### A. 自由文本 — Agent 先拟草稿，用户改/确认
| 字段 | key | 草稿来源 | 是否必填 |
|---|---|---|---|
| 产品名 | `productName` | 按品类+产地拟建议名（如「宜君玉米糁」） | 必填 |
| 规格 | `spec` | 给常见档位 250g/500g/1kg 让选，可自由填 | 必填 |
| 主视觉描述 | `mainVisual` | **核心字段**。Agent 按品类+风格拟一版画面描述，用户改 | 必填 |

### B. 点选预设 — `/images/options` 已有选项卡
| 字段 | key | 选法 | 预设选项 |
|---|---|---|---|
| 视觉风格 | `visualStyle` | 单选 | 摄影实拍 / 矢量插画·扁平 / 水彩·淡彩手绘 / 民俗绘画 / 剪纸·版画风 / 线描·简笔线条 / 帛画·水墨 / 像素插画 / 混合 |
| 主视觉类型 | `visualType` | 单选 | 插画 / 摄影实拍 / 文字图形 |
| 字体调性 | `fontTone` | 单选 | 清秀手写(轻松·亲和) / 厚重黑体(力量感·现代) / 毛笔书法(传统) / 圆体·卡通体(可爱·童趣) / 宋体(典雅·古典) |
| 版式骨架 | `layout` | 单选 | 上下分区型 / 居中型 / 满铺图 |
| Logo 位置 | `logoPos` | 单选 | 顶部居中 / 顶部左 / 顶部右 / 底部居中 / 中央 / 其他 |
| 结构惯例 | `structure` | **多选** | 开窗 / 腰封 / 全包 |
| 主色 | `mainColor` | 单选 | 红 / 橙 / 黄 / 绿 / 青 / 蓝 / 紫 / 多彩 / 中性 / 大地 |
| 包装形态 | `packForm` | 单选 | 袋装 / 普通盒装 / 礼盒装 / 瓶装 / 罐装 / 天然材质袋 / 箱装 / 异形 |

### C. 系统固定 / 上下文带入 — 不必逐次问用户
| 字段 | key | 取值方式 |
|---|---|---|
| 文化锚点 | `culturalAnchor` | 由县域绑定（宜君 → 宜君农民画），并触发参考图走图生图 |
| 参考图风格约束 | （写死进 prompt 尾部） | 农民画绘画风格 + 专业平面构图：主体突出、负空间充足、视觉层次清晰、装饰元素克制 |
| 品类大类 | `category` | 由产品名/选品类推断（干货农副 / 即食零食…） |

## 二、提问流程（4 步出图）

### Step 1 · 选品类（开场，`askUser`）
> message: "你想给什么产品做包装？"
> schema: `{ category: { type:"string", title:"品类", enum:["干货农副","即食零食","蜂蜜","茶","果干","大米","坚果","其他"] } }`

→ 拿到品类后即可推断 `category`，并预拟产品名草稿。

### Step 2 · 选视觉风格（`selectStyle` 图片点选）
- 调 `selectStyle`，传该品类对应的风格卡（含 `thumbnailUrl` + `promptAnchor`）。
- 宜君县域产品默认高亮「民俗绘画 / 宜君农民画」，并附参考图。
- 记住用户选中的 `visualStyle` 与 `promptAnchor`。

### Step 3 · 一次性收基础信息 + 版面预设（1~2 次 `askUser`，每项带凶狠默认）
**3a 基础信息**（Agent 先填草稿）：
> 字段：`productName`(预填建议名) / `spec`(enum 250g·500g·1kg + 自由填) / `packForm`(enum 默认「袋装」)
> 每个字段带中文 `title`，enum 用 `enumLabels` 给中文标签。

**3b 版面预设**（全部带「帮我选」兜底默认）：
> `visualType`(默认 插画) / `fontTone`(默认 厚重黑体) / `layout`(默认 上下分区型) / `logoPos`(默认 顶部右) / `structure`(多选，默认 全包·无开窗) / `mainColor`(默认 跟品类，玉米类→黄色系)
> 用户可一路点默认走完。

### Step 4 · 主视觉描述（Agent 拟草稿 → `askUser` 让用户改）
- Agent 按「品类 + 选中风格 + 主色」拟一段画面描述草稿，放进 `askUser` 的默认值。
  - 例（玉米糁）：「上部白色邮票内单株玉米简笔；下部红肚兜农民画男孩，旁边戴蝴蝶发饰女孩抱玉米穗，周围玉米地与彩色花卉」
- 用户改文字或直接「就这样」。存为 `mainVisual`。

### Step 5 · 确认 · 生图（`confirmBrief` → `generateImage`）
- `confirmBrief` 把所有字段汇成摘要卡 + 「🎨 开始生成」按钮。
- 用户点按钮（`confirmed:true`）后才调 `generateImage`，n=2~3，size=`1024x1536`，参考图 URL 进 `referenceImageUrls` 走图生图。

> 兜底：任一预设字段用户没选 → 取默认值，**绝不阻塞出图**。仅 `productName / spec / mainVisual` 三个自由文本字段为硬必填（但都已由 Agent 预填草稿，用户改即可）。

## 三、Prompt 组装模板（字段 → 输出）

```
根据参考图的宜君农民画包装设计风格，做一张宜君县域{category}{packForm}产品的正面包装设计。
参考图使用说明：不仅参考其农民画绘画风格，更要严格参考其专业平面设计构图——主体突出、负空间充足、视觉层次清晰、装饰元素克制。
产品名：{productName}
规格：{spec}
视觉风格：{visualStyle}
主视觉类型：{visualType}
文化锚点：{culturalAnchor}
字体调性：{fontTone}
版式：{layout}
Logo 位置：{logoPos}
结构：{structure}
主色：{mainColor}
主视觉描述：{mainVisual}
严格遵循参考图的农民画绘画风格与专业平面设计构图。只生成上述描述的元素，不要自行添加未提及的物体。
包装上需清晰印刷中文：主标题"{productName}"、规格"{spec}"，中文字体清晰、排版整齐、无错字、无多余文字。
```
